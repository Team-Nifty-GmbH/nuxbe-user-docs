# Payment Runs

In the **Payment Runs** section you combine multiple payments (direct debits or transfers) into batch payments.

## Overview

1. Navigate to **Accounting > Payment Runs**.

   ![Payment runs list](../screenshots/31-payment-runs-list.png)

2. The table shows all payment runs with status and number of included payments.

## Create a New Payment Run

1. Click **New**.
2. Select the type (direct debit or transfer).
3. Select the payments to be included in the run.
4. Click **Save**.

## Execute a Payment Run

1. Open the desired payment run.
2. Review the included payments.
3. Click **Execute** to submit the payment run to the bank.
