# Accounting

In the **Accounting** module you manage transactions, purchase invoices, dunning and payment runs.

## Overview

Navigate to **Accounting** via the sidebar. The submenu contains the following areas:

- **Commissions** - Commission management
- **Dunning** - Dunning for outstanding invoices
- **Purchase Invoices** - Recording and managing supplier invoices
- **Transactions** - Bank transactions and account movements
- **Transaction Assignments** - Assigning transactions to orders
- **Direct Debit** - Managing direct debit collections
- **Bank Transfer** - Processing bank transfers
- **Payment Runs** - Batch payment processing

## Transactions

The transactions overview shows all bank transactions in a tabular view.

![Transaction list with bank movements](../screenshots/15-transactions-list.png)

The table contains the following columns:

- **Bank Connection** - Name of the associated bank account
- **Value Date** - Date of the value posting
- **Amount** - Transaction amount
- **Counterpart Name** - Name of the payment recipient or sender
- **Purpose** - Description of the transaction
- **Bank Connection IBAN** - IBAN of your own account
- **Counterpart IBAN** - IBAN of the counterpart

## Features

- **Search** - Search transactions by purpose, counterpart or amount.
- **Add** - Manually enter new transactions.
- **Edit** - Click **Edit** to modify or assign a transaction.
