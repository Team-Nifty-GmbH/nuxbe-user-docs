# Payment Run Preview

The payment run preview lets you review all payments before executing a payment run.

## Open the Preview

1. Navigate to **Accounting > Payment Runs**.
2. Open an existing payment run or create a new one.
3. The preview is displayed with all included payments.

   ![Payment run preview](../screenshots/57-payment-run-preview.png)

## Columns

The table shows the following columns:

- **Invoice Number** - Number of the invoice to be paid
- **Invoice Date** - Date the invoice was issued
- **Contact > Customer Number** - Customer number of the recipient
- **Invoice Address > Name** - Name from the invoice address
- **Total Gross** - Gross amount of the invoice
- **Balance** - Remaining open balance
- **Commission** - Commission amount

## Search

Use the search field to find specific invoices within the payment run.

## Review and Execute

1. Review all listed payments carefully.
2. Select or deselect individual payments using the checkboxes.
3. Once satisfied, execute the payment run from the [Payment Runs](8-payment-runs.md) view.
