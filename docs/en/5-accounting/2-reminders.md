# Reminders

In the **Reminders** section you manage payment reminders for overdue invoices.

## Overview

1. Navigate to **Accounting > Reminders**.

   ![Reminders list](../screenshots/26-reminders-list.png)

2. The table shows all reminders with contact, invoice number, open amount and reminder level.

## Run Reminder Process

1. Click **Start Reminder Run** to automatically generate reminders for all overdue invoices.
2. Review the generated reminders.
3. Send reminders by email or print them.
