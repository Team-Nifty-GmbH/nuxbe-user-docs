# Media

In the **Media** module you manage files and documents centrally in Nuxbe.

## Overview

Navigate to **Media** via the sidebar. You will see the media library with an overview of all uploaded files.

![Media library](../screenshots/60-media-library.png)

## Toolbar

- **Sort** - Change the sort order of displayed files
- **Settings** - Configure display options using the gear icon

## Upload Files

1. Drag and drop files into the media library.
2. Alternatively, use the upload function to select files from your device.

## Organise with Folders

1. Create folders to organise your files by topic or project.
2. Move files between folders by dragging them.

## Preview Files

Click on a file to display a preview. Supported file types include images, PDFs and documents.

## Search Files

Use the search function to find files by name.
