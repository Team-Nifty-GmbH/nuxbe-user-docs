# Project Details

The detail view shows all information for a project and its tasks.

## Open the Detail View

1. Navigate to the [project list](1-manage-projects.md).
2. Click on the desired project.

   ![Project details](../screenshots/43-project-detail.png)

## Sections

The detail view is divided into several tabs:

### General

- **Name** - Project name
- **Status** - Current project status
- **Start Date / End Date** - Planned period
- **Progress** - Percentage progress
- **Responsible** - Project manager
- **Linked Order** - Associated order

### Tasks

Table of all assigned tasks with name, status and responsibility.

### Comments

Comments and notes for the project.

### Working Hours

Overview of recorded working hours for this project.

### Dashboard

Graphical representation of project progress.
