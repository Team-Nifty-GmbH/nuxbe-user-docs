# Task Details

The detail view shows all information for a single task.

## Open the Detail View

1. Navigate to the [task list](1-manage-tasks.md).
2. Click on the desired task.

   ![Task details](../screenshots/40-task-detail.png)

## Sections

- **Name** - Task name
- **Description** - Detailed description
- **Status** - Current status (open, in progress, done)
- **Priority** - Urgency level
- **Assigned To** - Responsible person
- **Project** - Linked project
- **Due Date** - Due date

### Comments

Add comments to document progress.

### Working Hours

View recorded working hours for this task.
