# Products

In the **Products** module you manage your product catalogue and serial numbers.

## Overview

Navigate to **Products** via the sidebar. The submenu contains:

- **Products** - Complete product catalogue
- **Serial Numbers** - Serial number management

## Product List

The product overview shows all products in a tabular view.

![Product list with all products](../screenshots/10-products-list.png)

The table contains typical columns such as:

- **Product Number** - Unique article number
- **Name** - Product name
- **Description** - Short description
- **Price** - Sales price
- **Is Active** - Product status

## Features

- **Search** - Search the product catalogue by name, number or description.
- **New Product** - Click **New** to create a new product.
- **Filter** - Filter products by various criteria.
