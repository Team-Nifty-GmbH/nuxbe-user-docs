# Addresses

Each contact can have multiple addresses, e.g. billing address, delivery address or branch locations.

## View Addresses

1. Open the [detail view](2-contact-detail.md) of a contact.
2. Switch to the **Addresses** tab.

   ![Address list of a contact](../screenshots/21-addresses-list.png)

3. The table shows all stored addresses with the following columns:
   - **Company** - Company name at this address
   - **Street** - Street and house number
   - **Postal Code** - Postal code
   - **City** - City
   - **Country** - Country
   - **Address Type** - Type of address (e.g. main address, delivery address)

## Add a New Address

1. Click **New** in the address tab.
2. Fill in the address fields.
3. Select the **Address Type**.
4. Click **Save**.

## Edit an Address

1. Click **Edit** next to the desired address.
2. Change the relevant fields.
3. Click **Save**.
