# API Tokens

Manage API tokens used by external services to authenticate with the system.

## Open API Tokens

1. Navigate to **Settings > Users & Permissions > API Tokens**.

   ![API tokens list](../screenshots/115-settings-api-tokens.png)

2. The table shows all tokens with the following columns:
   - **Name** - Token name
   - **Description** - Purpose of the token

## Create a Token

1. Click **New**.
2. Enter a name and an optional description.
3. Click **Save**. The generated token is shown once -- copy it immediately.

## Delete

- Click **Delete** to revoke a token. This cannot be undone.
