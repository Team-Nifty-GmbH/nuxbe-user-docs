# Serial Number Ranges

Configure the number ranges used when generating serial numbers for products.

## Open Serial Number Ranges

1. Navigate to **Settings > Products > Serial Number Ranges**.

   ![Serial number ranges list](../screenshots/107-settings-serial-number-ranges.png)

## Create a Range

1. Click **New**.
2. Define the prefix, current number and step.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to modify an existing range.
- Click **Delete** to remove a range.
