# Payment Types

Manage the payment methods available for orders, such as invoice, direct debit or prepayment.

## Open Payment Types

1. Navigate to **Settings > Orders > Payment Types**.

   ![Payment types list](../screenshots/95-settings-payment-types.png)

2. The table shows all payment types with the following columns:
   - **Name** - Payment type name
   - **Payment Reminder Day 1 / 2 / 3** - Days until each reminder level
   - **Is Active** - Whether the payment type is enabled
   - **Description** - Additional details for the payment type

## Create a Payment Type

1. Click **New**.
2. Enter a name and configure the reminder days.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to modify an existing payment type.
- Click **Delete** to remove a payment type.
