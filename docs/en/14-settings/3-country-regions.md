# Country Regions

Define regions or states within countries for more precise address management.

## Open Country Regions

1. Navigate to **Settings > General > Country Regions**.

   ![Country regions list](../screenshots/77-settings-country-regions.png)

2. The table shows all regions with the following columns:
   - **ID** - Region identifier
   - **Country** - The parent country
   - **Name** - Region name

## Create a Region

1. Click **New**.
2. Select a country and enter the region name.
3. Click **Save**.
