# Ticket Types

Manage the types of tickets available in the ticket module.

## Open Ticket Types

1. Navigate to **Settings > Tickets > Ticket Types**.

   ![Ticket types list](../screenshots/114-settings-ticket-types.png)

2. The table shows all ticket types with the following columns:
   - **ID** - Identifier
   - **Name** - Type name (e.g. Support, Bug, Feature Request, New Development)

## Create a Ticket Type

1. Click **New**.
2. Enter the name of the ticket type.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to rename a ticket type.
- Click **Delete** to remove a ticket type.
