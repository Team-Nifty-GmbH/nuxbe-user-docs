# Languages

Configure the languages available in the system for the user interface and translations.

## Open Languages

1. Navigate to **Settings > General > Languages**.

   ![Languages list](../screenshots/85-settings-languages.png)

2. The table shows all languages with the following columns:
   - **Name** - Language name
   - **ISO Name** - ISO language identifier
   - **Language Code** - Short code (e.g. en, de)

## Create a Language

1. Click **New**.
2. Enter the language name and codes.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to modify an existing language.
- Click **Delete** to remove a language.
