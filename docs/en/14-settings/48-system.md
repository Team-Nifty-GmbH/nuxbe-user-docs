# System

View technical details about the server environment including framework versions, PHP configuration, storage, cache, database and installed extensions.

## Open System

1. Navigate to **Settings > System > System**.

   ![System overview](../screenshots/109-settings-system.png)

2. The dashboard shows:
   - **Laravel Framework** - Version, environment, debug mode, timezone and locale
   - **PHP Configuration** - Version, memory limit, execution time and upload sizes
   - **Storage** - Free and total disk space, view cache size
   - **Queue** - Connection type, driver and queue size
   - **Sessions** - Driver, duration and secure flag
   - **Cache** - Cache driver and flush button
   - **Database** - Driver, host, port, open connections and table overview
   - **PHP Extensions** - List of all loaded extensions
   - **Scout** - Search index management

## Actions

- **Clear View Cache** - Remove compiled views.
- **Flush Queue** - Clear pending queue jobs.
- **Clear Cache** - Flush the application cache.
- **Clean Database** - Remove unused database entries.
