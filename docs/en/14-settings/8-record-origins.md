# Record Origins

Define the origins of data records to track where contacts and other entries come from.

## Open Record Origins

1. Navigate to **Settings > General > Record Origins**.

   ![Record origins list](../screenshots/104-settings-record-origins.png)

2. The table shows all record origins with the following columns:
   - **Name** - Origin name
   - **Model** - Related module (e.g. contact)
   - **Is Active** - Whether the origin is enabled

## Create a Record Origin

1. Click **New**.
2. Enter a name and select the model.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to modify an existing origin.
- Click **Delete** to remove an origin.
