# Currencies

Manage the currencies used for orders, price lists and accounting.

## Open Currencies

1. Navigate to **Settings > General > Currencies**.

   ![Currencies list](../screenshots/78-settings-currencies.png)

2. The table shows all currencies with the following columns:
   - **Name** - Currency name (e.g. Euro, Dollar)
   - **ISO** - ISO 4217 code (e.g. EUR, USD)
   - **Symbol** - Currency symbol
   - **ID** - Internal identifier

## Create a Currency

1. Click **New**.
2. Enter the name, ISO code and symbol.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to modify an existing currency.
- Click **Delete** to remove a currency.
