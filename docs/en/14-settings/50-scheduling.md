# Scheduling

View and manage scheduled tasks that run automatically in the background.

## Open Scheduling

1. Navigate to **Settings > System > Scheduling**.

   ![Scheduling list](../screenshots/106-settings-scheduling.png)

2. The list shows all registered scheduled commands with their frequency and last run time.
