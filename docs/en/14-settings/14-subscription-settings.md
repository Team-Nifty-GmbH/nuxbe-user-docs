# Subscription Settings

Configure defaults for recurring subscription orders, including the cancellation period and cancellation text.

## Open Subscription Settings

1. Navigate to **Settings > Orders > Subscription Settings**.

   ![Subscription settings](../screenshots/108-settings-subscription-settings.png)

2. The form contains:
   - **Default Cancellation Period** - Value and unit (e.g. days) before a subscription can be cancelled
   - **Cancellation Text** - Rich-text template used in cancellation notices

## Save Changes

1. Adjust the values as needed.
2. Click **Save**.
