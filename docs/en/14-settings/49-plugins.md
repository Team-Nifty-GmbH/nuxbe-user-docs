# Plugins

Manage installed plugins that extend the functionality of the system.

## Open Plugins

1. Navigate to **Settings > System > Plugins**.

   ![Plugins list](../screenshots/97-settings-plugins.png)

## Install or Update

1. Click **New** to install a new plugin or click **Update** on an existing one.
2. Follow the on-screen instructions.

## Enable or Disable

- Toggle a plugin's active state to enable or disable it without uninstalling.
