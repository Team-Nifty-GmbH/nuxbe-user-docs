# Price Lists

Manage price lists used to define product pricing for different customer groups or sales channels.

## Open Price Lists

1. Navigate to **Settings > Orders > Price Lists**.

   ![Price lists](../screenshots/98-settings-price-lists.png)

2. The table shows all price lists with the following columns:
   - **Name** - Price list name
   - **Price List Code** - Short code identifier
   - **Net** - Whether prices are net
   - **Is Default** - Whether this is the default price list
   - **ID** - Internal identifier

## Create a Price List

1. Click **New**.
2. Enter a name and code, and set the net and default flags.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to modify an existing price list.
- Click **Delete** to remove a price list.
