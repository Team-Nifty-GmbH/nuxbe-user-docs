# Core Settings

Configure fundamental application settings such as the salutation style, installation status and licence key.

## Open Core Settings

1. Navigate to **Settings > System > Core Settings**.

   ![Core settings](../screenshots/75-settings-core-settings.png)

2. The form contains:
   - **Formal Salutation** - Use formal address forms throughout the system
   - **Installation Complete** - Marks the initial setup as finished
   - **Licence Key** - Your application licence key

## Save Changes

1. Adjust the toggles or enter the licence key.
2. Click **Save**.
