# Print Jobs

View and manage print jobs sent to connected printers.

## Open Print Jobs

1. Navigate to **Settings > System > Print Jobs**.

   ![Print jobs list](../screenshots/99-settings-print-jobs.png)

2. The table shows all print jobs with their status, printer and creation time.
