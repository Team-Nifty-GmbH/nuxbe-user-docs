# Address Types

Manage the types of addresses that can be assigned to contacts, such as billing or shipping.

## Open Address Types

1. Navigate to **Settings > Contacts > Address Types**.

   ![Address types list](../screenshots/72-settings-address-types.png)

2. The table shows all address types with the following columns:
   - **Name** - Type name
   - **Address Type Code** - Short code
   - **Is Locked** - Whether the type is protected from deletion
   - **Is Unique** - Whether only one address of this type is allowed per contact

## Create an Address Type

1. Click **New**.
2. Enter a name and code.
3. Click **Save**.
