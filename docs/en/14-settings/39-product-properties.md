# Product Properties

Define custom properties that can be attached to products to store additional attributes.

## Open Product Properties

1. Navigate to **Settings > Products > Product Properties**.

   ![Product properties list](../screenshots/102-settings-product-properties.png)

## Create a Property

1. Click **New**.
2. Enter a name and select the value type.
3. Click **Save**.

## Edit or Delete

- Click **Edit** to modify an existing property.
- Click **Delete** to remove a property.
