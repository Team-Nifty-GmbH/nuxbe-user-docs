# Order Positions List

The order positions list shows all positions across all orders in a single table view.

## Open the List

1. Navigate to **Orders > Order Positions**.

   ![Order positions list](../screenshots/56-order-positions-list.png)

## Columns

The table shows the following columns:

- **Product Number** - Article number of the product
- **Order > Order Number** - Number of the parent order
- **Order > Invoice Number** - Invoice number of the parent order
- **Name** - Product or position name
- **Quantity** - Ordered quantity
- **Total Net** - Net amount of the position
- **Order > Invoice Address > Company** - Company from the invoice address
- **Description** - Position description

## Search and Filter

- Use the search field to search across all positions.
- Use the filter row below the column headers to filter by product number, order number, company or other criteria.
