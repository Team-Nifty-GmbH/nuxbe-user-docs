# Absence Request Details

The detail view shows all information for a single absence request.

## Open the Detail View

1. Navigate to the [absence requests list](6-absence-requests.md).
2. Click on the desired request.

   ![Absence request detail view](../screenshots/65-absence-request-detail.png)

## Tabs

The detail view is divided into several tabs:

### General

The **General** tab contains the basic request data:

- **Absence Type** - Type of absence (e.g. holiday, sick leave)
- **Day Section** - Full day or half day
- **Start Date** - First day of absence
- **End Date** - Last day of absence
- **Substitute** - Selected substitute employee
- **Reason** - Reason for the absence
- **Substitute Note** - Notes for the substitute

### Approval

View the current approval status and approve or reject the request.

### Employee Days

Overview of the employee days affected by this absence.

### Comments

Add comments to the request.

### Activities

History of changes and events related to the request.

## Approval Status

The header shows the current status of the request:

- **Approved** - The request has been approved
- **Requested** - The request is pending review
- **Rejected** - The request has been rejected

## Save Changes

Click **Save** to apply any changes to the request.
