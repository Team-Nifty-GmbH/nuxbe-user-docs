# Working Hours

In the **Working Hours** section you view all recorded working hours.

## Overview

1. Navigate to **Human Resources > Working Hours**.

   ![Working hours list](../screenshots/37-working-hours-list.png)

2. The table shows all time entries with date, employee, start time, end time and duration.

## Filter Working Hours

- Filter by employee, time period or project.
- Use the filter row below the column headers.
