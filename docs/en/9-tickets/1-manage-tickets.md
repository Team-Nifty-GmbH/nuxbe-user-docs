# Manage Tickets

The ticket list shows all support tickets and their current processing status.

## Open the Ticket List

1. Click **Tickets** in the sidebar.

   ![Ticket list](../screenshots/41-tickets-list.png)

2. The table shows all tickets with the following columns:
   - **Ticket Number** - Unique number
   - **Ticket Type** - Type of ticket (Bug, Feature Request, Support, etc.)
   - **User** - Ticket creator
   - **Title** - Subject of the ticket
   - **Status** - Current processing status
   - **Created At** - Creation date

## Filter Tickets

- Use the search field to find tickets by title or number.
- Filter by status using the dropdown in the status column.
- Use **Saved Filters** for frequently used filter sets.

## Create a New Ticket

1. Click **New**.
2. Select the **Ticket Type**.
3. Enter title and description.
4. Click **Save**.
