# Aufgaben verwalten

In der Aufgabenliste sehen Sie alle Aufgaben mit Status, Priorität und Zuständigkeiten auf einen Blick.

## Aufgabenliste öffnen

1. Klicken Sie in der Sidebar auf **Aufgaben**.

   ![Aufgabenliste](../screenshots/39-aufgaben-liste.png)

2. Die Tabelle zeigt alle Aufgaben mit folgenden Spalten:
   - **Name** - Bezeichnung der Aufgabe
   - **Beschreibung** - Kurzvorschau der Aufgabenbeschreibung
   - **Priorität** - Dringlichkeitsstufe, farblich hervorgehoben (Niedrig, Normal, Hoch, Dringend)
   - **Status** - Aktueller Bearbeitungsstand (Offen, In Bearbeitung, Erledigt, Abgebrochen)
   - **Zugewiesen an** - Verantwortliche Personen (es können mehrere Mitarbeiter zugewiesen sein)
   - **Projekt** - Zugehöriges Projekt, sofern verknüpft
   - **Fällig am** - Fälligkeitsdatum; überfällige Aufgaben werden optisch gekennzeichnet
   - **Zeitbudget** - Geplantes Zeitbudget in Stunden
   - **Erfasste Zeit** - Bereits auf diese Aufgabe gebuchte Arbeitszeit

## Aufgaben filtern

### Suchfeld

Nutzen Sie das Suchfeld oberhalb der Tabelle, um Aufgaben nach Name oder Beschreibung zu durchsuchen. Die Ergebnisse werden in Echtzeit gefiltert.

### Gespeicherte Filter

Verwenden Sie **Gespeicherte Filter** für häufig verwendete Filtereinstellungen. Typische vordefinierte Filter sind:

- **Meine Aufgaben** - Zeigt nur Aufgaben, die Ihnen zugewiesen sind
- **Überfällige Aufgaben** - Zeigt Aufgaben, deren Fälligkeitsdatum überschritten ist
- **Offene Aufgaben** - Zeigt alle Aufgaben mit dem Status Offen oder In Bearbeitung

Sie können eigene Filter erstellen und speichern, indem Sie die gewünschten Filterkriterien einstellen und auf **Filter speichern** klicken.

### Spaltenfilter

Nutzen Sie die Filterzeilen unterhalb der Spaltenüberschriften, um die Tabelle gezielt nach einzelnen Spalten einzuschränken, beispielsweise nach einem bestimmten Status oder einer bestimmten Priorität.

## Neue Aufgabe anlegen

1. Klicken Sie auf die Schaltfläche **Neu** oben rechts.
2. Füllen Sie die folgenden Felder aus:
   - **Name** - Geben Sie eine aussagekräftige Bezeichnung für die Aufgabe ein.
   - **Beschreibung** - Beschreiben Sie die Aufgabe im Richtext-Editor. Sie können Formatierungen, Listen und Links verwenden.
   - **Priorität** - Wählen Sie die Dringlichkeit (Niedrig, Normal, Hoch, Dringend).
   - **Zugewiesen an** - Wählen Sie einen oder mehrere Mitarbeiter aus, die für die Aufgabe verantwortlich sind.
   - **Projekt** - Ordnen Sie die Aufgabe optional einem bestehenden Projekt zu.
   - **Fällig am** - Setzen Sie ein Fälligkeitsdatum, bis zu dem die Aufgabe erledigt sein soll.
   - **Zeitbudget** - Geben Sie optional ein Zeitbudget in Stunden ein, um den geplanten Aufwand festzuhalten.
   - **Übergeordnete Aufgabe** - Wählen Sie optional eine übergeordnete Aufgabe, um eine Aufgabenhierarchie aufzubauen. Die neue Aufgabe wird dann als Unteraufgabe angelegt.
3. Klicken Sie auf **Speichern**, um die Aufgabe anzulegen.

Die neue Aufgabe wird mit dem Status **Offen** erstellt und erscheint in der Aufgabenliste.

## Zeiterfassung aus der Liste

Klicken Sie auf die Schaltfläche **Zeiterfassung** in der Aufgabenliste, um direkt eine Zeiterfassung für eine Aufgabe zu starten. Ein laufender Timer wird in der Oberfläche angezeigt und kann jederzeit gestoppt werden. Die erfasste Zeit wird automatisch der jeweiligen Aufgabe zugeordnet.
