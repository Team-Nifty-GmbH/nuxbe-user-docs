# Erste Schritte

Dieser Abschnitt führt Sie durch die grundlegenden Funktionen von Nuxbe. Sie lernen, wie Sie sich anmelden, die Benutzeroberfläche bedienen, Ihr Dashboard einrichten und Ihr Benutzerprofil anpassen.

## Inhalt

- [Anmeldung](1-anmeldung.md) - An- und Abmeldung, Passwort zurücksetzen, Magic Link
- [Navigation](2-navigation.md) - Sidebar, globale Suche, Benutzermenü, Favoriten und häufig besuchte Seiten
- [Dashboard](3-dashboard.md) - Widgets, Tabs, Zeitraumfilter und Bearbeitungsmodus
- [Mein Profil](4-mein-profil.md) - Persönliche Daten, Sprache, Passwort, Benachrichtigungskanäle und Push-Benachrichtigungen
- [Merklisten](5-merklisten.md) - Produkte auf Merklisten speichern und verwalten
