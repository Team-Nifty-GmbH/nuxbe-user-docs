# Anmeldung und Abmeldung

## Anmeldung

Öffnen Sie die Nuxbe-Anwendung in Ihrem Browser. Auf der Anmeldeseite stehen Ihnen zwei Wege zur Verfügung:

### Anmeldung mit E-Mail und Passwort

1. Geben Sie Ihre **E-Mail-Adresse** ein.
2. Geben Sie Ihr **Passwort** ein.
3. Optional: Setzen Sie das Häkchen bei **Angemeldet bleiben**, um Ihre Sitzung über das Schließen des Browsers hinaus zu erhalten.
4. Klicken Sie auf **Anmelden**.

### Anmeldung per Magic Link

Wenn Sie kein Passwort eingeben möchten, können Sie sich per Magic Link anmelden:

1. Geben Sie Ihre **E-Mail-Adresse** ein.
2. Klicken Sie auf **Magic Link senden**.
3. Prüfen Sie Ihr E-Mail-Postfach. Sie erhalten einen Link, mit dem Sie sich ohne Passwort direkt anmelden können.

> **Hinweis:** Der Magic Link ist aus Sicherheitsgründen nur 15 Minuten gültig. Fordern Sie bei Bedarf einen neuen an.

## Erste Anmeldung

Wenn Sie als neuer Benutzer angelegt wurden und noch kein Passwort haben:

1. Sie erhalten eine E-Mail mit einem Einladungslink oder nutzen die **Magic Link**-Funktion.
2. Klicken Sie auf den Link in der E-Mail. Sie werden direkt angemeldet.
3. Navigieren Sie zu [Mein Profil](4-mein-profil.md), um ein persönliches Passwort zu vergeben.
4. Geben Sie im Feld **Neues Passwort** Ihr gewünschtes Passwort ein und bestätigen Sie es im Feld **Passwort wiederholen**.
5. Klicken Sie auf **Speichern**.

Ab sofort können Sie sich mit Ihrer E-Mail-Adresse und dem neuen Passwort anmelden.

> **Hinweis:** Beim Bearbeiten anderer Benutzer in der Verwaltung können die Felder **Neues Passwort** und **Passwort wiederholen** ebenfalls sichtbar sein. Lassen Sie diese Felder leer, wenn Sie nur andere Daten des Benutzers ändern möchten (z. B. Name, Zuweisungen). Wenn das Feld **Neues Passwort** gefüllt ist, müssen Sie das Passwort auch im Feld **Passwort wiederholen** bestätigen -- andernfalls erscheint eine Fehlermeldung.

## Passwort zurücksetzen

Falls Sie Ihr Passwort vergessen haben:

1. Klicken Sie auf der Anmeldeseite auf **Passwort zurücksetzen**.
2. Geben Sie Ihre E-Mail-Adresse ein.
3. Klicken Sie auf **Link senden**.
4. Öffnen Sie die E-Mail und klicken Sie auf den enthaltenen Link.
5. Vergeben Sie ein neues Passwort und bestätigen Sie es.
6. Sie werden anschließend zur Anmeldeseite weitergeleitet.

## Sitzungsverwaltung

Ihre Browser-Sitzung läuft nach einer bestimmten Zeit der Inaktivität automatisch ab. Nach dem Ablauf werden Sie zur Anmeldeseite weitergeleitet und müssen sich erneut anmelden.

> **Tipp:** Aktivieren Sie bei der Anmeldung die Option **Angemeldet bleiben**, um die Sitzungsdauer zu verlängern. So müssen Sie sich nicht bei jedem Browserstart erneut anmelden.

> **Wichtig:** Wenn Sie längere Zeit an einem Auftrag, Kontakt oder einer anderen Seite arbeiten, speichern Sie Ihre Änderungen regelmäßig zwischendurch. Bei einem Sitzungsablauf gehen nicht gespeicherte Änderungen verloren.

## Abmeldung

1. Klicken Sie oben links in der Sidebar auf Ihren Benutzernamen unterhalb von **Angemeldet als:**.

   ![Benutzermenü mit Optionen Mein Profil und Abmelden](../screenshots/05-benutzermenue.png)

2. Klicken Sie auf **Abmelden**.

Sie werden zur Anmeldeseite weitergeleitet. Ihre Sitzung wird dabei vollständig beendet.
