# Produktoptionsgruppen

Unter **Einstellungen > Produkte > Produktoptionsgruppen** verwalten Sie Gruppen von Produktoptionen, die zur Erstellung von Produktvarianten verwendet werden.

Produktoptionsgruppen ermöglichen es Ihnen, verschiedene Ausprägungen eines Produkts zu definieren. Beispiele sind Farben (Rot, Blau, Grün), Größen (S, M, L, XL) oder Materialien (Holz, Metall, Kunststoff). Aus den Kombinationen dieser Optionen entstehen dann konkrete Produktvarianten.

## Übersicht

1. Navigieren Sie zu **Einstellungen > Produkte > Produktoptionsgruppen**.

   ![Produktoptionsgruppen verwalten](../screenshots/101-einstellungen-produktoptionsgruppen.png)

2. Die Tabelle zeigt alle Produktoptionsgruppen in der Übersicht.

### Angezeigte Spalten

- **Name** - Die Bezeichnung der Optionsgruppe (z. B. "Farbe", "Größe")
- **Optionen** - Die zugehörigen Optionswerte der Gruppe

Sie können die Liste nach Namen durchsuchen.

## Produktoptionsgruppe anlegen

Um eine neue Produktoptionsgruppe zu erstellen:

1. Klicken Sie auf die Schaltfläche **Neu**.
2. Geben Sie im Feld **Name** die Bezeichnung der Gruppe ein (z. B. "Farbe").
3. Fügen Sie die gewünschten Optionen hinzu:
   - Klicken Sie auf **Option hinzufügen**
   - Geben Sie den Namen der Option ein (z. B. "Rot", "Blau", "Grün")
   - Wiederholen Sie diesen Schritt für alle benötigten Optionen
4. Klicken Sie auf **Speichern**, um die Gruppe zu erstellen.

### Hinweise zum Anlegen

- Der **Name** der Gruppe sollte die Art der Variation beschreiben (z. B. "Größe", nicht "S/M/L").
- Jede Gruppe kann beliebig viele Optionen enthalten.
- Die Optionen werden später bei der Produktverwaltung zur Variantenerstellung kombiniert.

### Beispiel: Farbvarianten

Wenn Sie ein T-Shirt in verschiedenen Farben anbieten möchten:

1. Erstellen Sie eine Optionsgruppe mit dem Namen "Farbe"
2. Fügen Sie die Optionen "Weiß", "Schwarz", "Rot", "Blau" hinzu
3. Diese Optionsgruppe kann dann bei der Produktanlage ausgewählt werden

## Produktoptionsgruppe bearbeiten

Um eine bestehende Produktoptionsgruppe zu ändern:

1. Klicken Sie auf die gewünschte Optionsgruppe in der Tabelle.
2. Bearbeiten Sie den **Namen** der Gruppe oder ändern Sie die zugehörigen Optionen.
3. Sie können neue Optionen hinzufügen oder bestehende entfernen.
4. Klicken Sie auf **Speichern**, um die Änderungen zu übernehmen.

## Produktoptionsgruppe löschen

Optionsgruppen können gelöscht werden, sofern sie nicht mehr verwendet werden:

1. Öffnen Sie die zu löschende Optionsgruppe.
2. Klicken Sie auf **Löschen**.
3. Bestätigen Sie den Löschvorgang.

> **Hinweis:** Produktoptionsgruppen, die bereits Produkten zugeordnet sind, können nicht gelöscht werden. Sie müssen zunächst die Zuordnung bei den betroffenen Produkten entfernen.

## Verwendung von Produktoptionsgruppen

Produktoptionsgruppen werden verwendet, um Produktvarianten zu erstellen:

1. **Bei der Produktanlage** - Wählen Sie eine oder mehrere Optionsgruppen aus
2. **Automatische Variantenerstellung** - Das System erstellt automatisch alle möglichen Kombinationen
3. **Individuelle Preise** - Jede Variante kann einen eigenen Preis haben
4. **Separate Lagerbestände** - Varianten werden separat im Lager verwaltet

### Beispiel: Kombination mehrerer Gruppen

Sie verkaufen T-Shirts in verschiedenen Farben und Größen:

- Optionsgruppe "Farbe" mit den Optionen: Weiß, Schwarz, Rot
- Optionsgruppe "Größe" mit den Optionen: S, M, L, XL

Bei der Produktanlage werden automatisch 12 Varianten erstellt (3 Farben × 4 Größen).

> **Empfehlung:** Planen Sie Ihre Optionsgruppen sorgfältig. Nachträgliche Änderungen können umfangreiche Anpassungen bei bereits erstellten Produktvarianten erforderlich machen.
