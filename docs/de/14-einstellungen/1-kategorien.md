# Kategorien

Kategorien dienen zur Klassifizierung von Datensätzen in verschiedenen Modulen. Sie können beispielsweise Kontakte, Produkte, Artikel oder andere Entitäten in thematische Gruppen einteilen und so für eine bessere Übersichtlichkeit sorgen.

## Übersicht

1. Navigieren Sie zu **Einstellungen > Allgemein > Kategorien**.

   ![Kategorien verwalten](../screenshots/74-einstellungen-kategorien.png)

2. Die Tabelle zeigt alle angelegten Kategorien mit folgenden Spalten:
   - **Name** - Bezeichnung der Kategorie
   - **Modul** - Das Modul, dem die Kategorie zugeordnet ist (z. B. Kontakte, Produkte, Artikel)
   - **Sortierung** - Reihenfolge der Anzeige in den Auswahllisten
   - **Aktiv** - Status der Kategorie

## Kategorie anlegen

1. Klicken Sie auf **Neu**.
2. Füllen Sie die folgenden Felder aus:
   - **Name** - Bezeichnung der Kategorie (z. B. "VIP-Kunde", "Standardprodukt")
   - **Modul** - Wählen Sie das Modul, für das die Kategorie gelten soll
   - **Übergeordnete Kategorie** - Optional: Wählen Sie eine übergeordnete Kategorie für hierarchische Strukturen
   - **Sortierung** - Reihenfolge innerhalb der Kategorienliste (niedrigere Zahlen erscheinen zuerst)
   - **Aktiv** - Setzen Sie den Status der Kategorie
3. Klicken Sie auf **Speichern**.

## Kategorie bearbeiten

1. Klicken Sie auf eine Kategorie in der Liste.
2. Ändern Sie die gewünschten Felder.
3. Klicken Sie auf **Speichern**.

## Hierarchische Kategorien

Sie können Kategorien hierarchisch strukturieren, indem Sie eine übergeordnete Kategorie auswählen. Dies ermöglicht eine mehrstufige Gliederung wie z. B.:
- Elektronik
  - Computer
  - Smartphones
- Dienstleistungen
  - Beratung
  - Support

## Kategorie löschen

1. Klicken Sie auf eine Kategorie in der Liste.
2. Klicken Sie auf **Löschen**.
3. Bestätigen Sie die Löschung.

> **Hinweis:** Kategorien, die bereits Datensätzen zugewiesen sind, können nicht gelöscht werden. Deaktivieren Sie die Kategorie stattdessen, um sie aus den Auswahllisten zu entfernen.
