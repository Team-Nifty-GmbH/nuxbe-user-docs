# Sprachen

Unter **Einstellungen > Allgemein > Sprachen** konfigurieren Sie die im System verfügbaren Sprachen. Sprachen werden für die Benutzeroberfläche, mehrsprachige Inhalte und die Zuordnung zu Ländern verwendet.

## Übersicht

1. Navigieren Sie zu **Einstellungen > Allgemein > Sprachen**.

   ![Sprachen verwalten](../screenshots/85-einstellungen-sprachen.png)

2. Die Tabelle zeigt alle Sprachen mit folgenden Spalten:
   - **Name** - Bezeichnung der Sprache (z. B. "Deutsch", "English")
   - **ISO Name** - Internationaler Name der Sprache
   - **Sprachcode** - ISO-639-Sprachcode (z. B. "de", "en", "fr")
   - **Standard** - Kennzeichnet die Standardsprache des Systems

## Sprache anlegen

1. Klicken Sie auf **Neu**.
2. Füllen Sie die folgenden Felder aus:
   - **Name** - Bezeichnung der Sprache (z. B. "Deutsch", "Français")
   - **ISO Name** - Internationaler Name (z. B. "German", "French")
   - **Sprachcode** - ISO-639-Sprachcode (Pflichtfeld, eindeutig, z. B. "de", "fr", "en")
   - **Standard** - Markiert diese Sprache als Standardsprache für neue Datensätze
3. Klicken Sie auf **Speichern**.

## Sprache bearbeiten

1. Klicken Sie auf eine Sprache in der Liste.
2. Ändern Sie die gewünschten Felder.
3. Klicken Sie auf **Speichern**.

## Sprache löschen

1. Klicken Sie auf eine Sprache in der Liste.
2. Klicken Sie auf **Löschen**.
3. Bestätigen Sie die Löschung.

## Häufig verwendete Sprachcodes

- **de** - Deutsch
- **en** - English
- **fr** - Français
- **it** - Italiano
- **es** - Español

## Standardsprache

Die als Standard markierte Sprache wird für die Benutzeroberfläche und als Fallback für mehrsprachige Inhalte verwendet. Sie sollten genau eine Sprache als Standard definieren.

> **Hinweis:** Sprachen, die bereits Benutzern oder Ländern zugeordnet sind, können nicht gelöscht werden. Der Sprachcode sollte dem ISO-639-Standard entsprechen.
