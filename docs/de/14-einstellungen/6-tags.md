# Tags

Unter **Einstellungen > Allgemein > Tags** verwalten Sie Tags zur flexiblen Kennzeichnung von Datensätzen in verschiedenen Modulen. Tags ermöglichen eine schnelle visuelle Kategorisierung und Filterung ohne starre Kategorienstrukturen.

## Übersicht

1. Navigieren Sie zu **Einstellungen > Allgemein > Tags**.

   ![Tags verwalten](../screenshots/110-einstellungen-tags.png)

2. Die Tabelle zeigt alle Tags mit folgenden Spalten:
   - **Name** - Bezeichnung des Tags (z. B. "Dringend", "VIP", "Newsletter")
   - **Slug** - URL-freundliche Variante des Namens (automatisch generiert)
   - **Typ** - Modulzuordnung für spezifische Tags (optional)
   - **Farbe** - Farbcode zur visuellen Kennzeichnung
   - **Sortierung** - Reihenfolge der Anzeige

## Tag anlegen

1. Klicken Sie auf **Neu**.
2. Füllen Sie die folgenden Felder aus:
   - **Name** - Bezeichnung des Tags (z. B. "Wichtig", "Archiviert", "Follow-up")
   - **Typ** - Optional: Modulzuordnung für spezifische Tags (z. B. "contact", "product")
   - **Farbe** - Wählen Sie eine Farbe zur visuellen Kennzeichnung (z. B. "#ff0000" für Rot)
   - **Sortierung** - Reihenfolge innerhalb der Tag-Liste
3. Klicken Sie auf **Speichern**.

## Tag bearbeiten

1. Klicken Sie auf einen Tag in der Liste.
2. Ändern Sie die gewünschten Felder.
3. Klicken Sie auf **Speichern**.

## Tag löschen

1. Klicken Sie auf einen Tag in der Liste.
2. Klicken Sie auf **Löschen**.
3. Bestätigen Sie die Löschung.

## Verwendung von Tags

Tags können in verschiedenen Modulen verwendet werden:
- **Kontakte:** Kennzeichnung von VIP-Kunden, Newsletter-Abonnenten, etc.
- **Aufträge:** Markierung dringender Aufträge, spezielle Konditionen
- **Produkte:** Sonderangebote, neue Artikel, auslaufende Produkte
- **Projekte:** Status-Tags, Prioritäten, Team-Zuordnungen

## Farbcodes

Verwenden Sie Farben für eine schnelle visuelle Orientierung:
- **Rot (#ff0000):** Dringend, Wichtig, Warnung
- **Grün (#00ff00):** Erledigt, Genehmigt, Aktiv
- **Blau (#0000ff):** Information, Standard
- **Gelb (#ffff00):** In Bearbeitung, Warnung
- **Grau (#808080):** Archiviert, Inaktiv

> **Hinweis:** Tags sind flexibler als Kategorien und können einem Datensatz mehrfach zugewiesen werden. Der Slug wird automatisch aus dem Namen generiert und dient zur eindeutigen Identifikation.
