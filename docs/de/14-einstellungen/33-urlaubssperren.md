# Urlaubssperren

Unter **Einstellungen > Personalwesen > Urlaubssperren** legen Sie Zeiträume fest, in denen kein Urlaub genommen werden darf. Dies ist besonders wichtig für betriebskritische Phasen wie Inventur, Hauptsaison oder wichtige Projektphasen.

## Übersicht

1. Navigieren Sie zu **Einstellungen > Personalwesen > Urlaubssperren**.

   ![Urlaubssperren verwalten](../screenshots/118-einstellungen-urlaubssperren.png)

2. Die Tabelle zeigt alle Urlaubssperren mit folgenden Informationen:
   - **Name** - Bezeichnung der Urlaubssperre
   - **Zeitraum** - Von-Bis Datum
   - **Beschreibung** - Begründung für die Urlaubssperre
   - **Betroffene** - Standorte, Abteilungen oder einzelne Mitarbeiter
   - **Status** - Aktiv oder inaktiv

## Urlaubssperre anlegen

1. Klicken Sie auf **Neu**.
2. Füllen Sie die Pflichtfelder aus:
   - **Name** - Geben Sie eine Bezeichnung ein (z. B. "Inventur 2026")
   - **Startdatum** - Wählen Sie den Beginn der Urlaubssperre
   - **Enddatum** - Wählen Sie das Ende der Urlaubssperre
3. Optional können Sie weitere Informationen hinterlegen:
   - **Beschreibung** - Geben Sie eine Begründung für die Urlaubssperre ein
4. Legen Sie fest, für wen die Urlaubssperre gilt:
   - **Standorte** - Wählen Sie betroffene Standorte aus
   - **Abteilungen** - Wählen Sie betroffene Abteilungen aus
   - **Mitarbeiter** - Wählen Sie einzelne betroffene Mitarbeiter aus
   - **Aktiv** - Legen Sie fest, ob die Urlaubssperre aktiv ist
5. Klicken Sie auf **Speichern**.

## Urlaubssperre bearbeiten

1. Klicken Sie auf eine Urlaubssperre in der Liste.
2. Passen Sie die gewünschten Felder an.
3. Klicken Sie auf **Speichern**.

## Urlaubssperre löschen

1. Klicken Sie auf eine Urlaubssperre in der Liste.
2. Klicken Sie auf **Löschen**.
3. Bestätigen Sie den Löschvorgang.

## Geltungsbereich festlegen

Sie können Urlaubssperren für verschiedene Bereiche definieren:

**Standortbezogen:**
Wählen Sie einen oder mehrere Standorte aus. Alle Mitarbeiter dieser Standorte können in dem Zeitraum keinen Urlaub beantragen.

**Abteilungsbezogen:**
Wählen Sie eine oder mehrere Abteilungen aus. Nur Mitarbeiter dieser Abteilungen sind von der Sperre betroffen.

**Mitarbeiterbezogen:**
Wählen Sie einzelne Mitarbeiter aus. Nur diese spezifischen Mitarbeiter können keinen Urlaub beantragen.

**Kombiniert:**
Sie können auch mehrere Bereiche kombinieren, um die Urlaubssperre genau zu definieren.

Wenn Sie keine Auswahl treffen, gilt die Urlaubssperre für das gesamte Unternehmen.

## Auswirkungen auf Urlaubsanträge

Wenn ein Mitarbeiter versucht, Urlaub in einem gesperrten Zeitraum zu beantragen, erhält er eine Fehlermeldung. Bereits genehmigte Urlaubsanträge vor Anlage der Urlaubssperre bleiben bestehen.

Administratoren können bei Bedarf dennoch Urlaubsanträge für gesperrte Zeiträume anlegen oder genehmigen.

## Beispiele für Urlaubssperren

**Jahresinventur:**
- Zeitraum: 15.01.2026 - 20.01.2026
- Betroffen: Alle Mitarbeiter im Lager
- Begründung: Jährliche Bestandsaufnahme

**Hauptsaison Einzelhandel:**
- Zeitraum: 01.12.2026 - 24.12.2026
- Betroffen: Alle Mitarbeiter im Verkauf
- Begründung: Weihnachtsgeschäft

**Projektabschluss:**
- Zeitraum: 01.03.2026 - 31.03.2026
- Betroffen: Projektteam XY (einzelne Mitarbeiter)
- Begründung: Kritische Projektphase

> **Hinweis:** Kommunizieren Sie Urlaubssperren rechtzeitig an Ihre Mitarbeiter, damit diese ihre Urlaubsplanung entsprechend anpassen können. Aktivieren Sie Urlaubssperren nur, wenn sie wirklich benötigt werden, um die Flexibilität der Mitarbeiter nicht unnötig einzuschränken.
