# Abwesenheitsarten

Unter **Einstellungen > Personalwesen > Abwesenheitsarten** verwalten Sie die verschiedenen Arten von Abwesenheiten wie Urlaub, Krankheit, Sonderurlaub oder unbezahlten Urlaub. Jede Abwesenheitsart kann individuell konfiguriert werden und bestimmt, wie die Abwesenheit erfasst und angerechnet wird.

## Übersicht

1. Navigieren Sie zu **Einstellungen > Personalwesen > Abwesenheitsarten**.

   ![Abwesenheitsarten verwalten](../screenshots/69-einstellungen-abwesenheitsarten.png)

2. Die Tabelle zeigt alle Abwesenheitsarten mit folgenden Informationen:
   - **Name** - Bezeichnung der Abwesenheitsart
   - **Code** - Kurzbezeichnung
   - **Farbe** - Farbliche Kennzeichnung im Kalender
   - **Mitarbeiter darf anlegen** - Ob Mitarbeiter selbst Anträge stellen dürfen
   - **Status** - Aktiv oder inaktiv

## Abwesenheitsart anlegen

1. Klicken Sie auf **Neu**.
2. Füllen Sie die Pflichtfelder aus:
   - **Name** - Geben Sie die Bezeichnung ein (z. B. "Urlaub", "Krankheit")
   - **Code** - Kurzbezeichnung für die Abwesenheitsart
   - **Farbe** - Wählen Sie eine Farbe für die Kalenderansicht
3. Konfigurieren Sie die Optionen:
   - **Mitarbeiter darf anlegen** - Legen Sie fest, ob Mitarbeiter diese Abwesenheitsart selbst beantragen dürfen (Ja, Nein oder Mit Genehmigung)
   - **Prozentuale Anrechnung** - Geben Sie an, zu wie viel Prozent die Abwesenheit angerechnet wird (100% = volle Anrechnung)
   - **Betrifft Urlaub** - Aktivieren Sie dies, wenn die Abwesenheit vom Urlaubskontingent abgezogen werden soll
   - **Betrifft Krankheit** - Aktivieren Sie dies, wenn es sich um eine Krankheitsart handelt
   - **Betrifft Überstunden** - Aktivieren Sie dies, wenn die Abwesenheit Überstunden beeinflusst
   - **Abwesenheitsrichtlinien** - Wählen Sie die zugehörigen Richtlinien aus
   - **Aktiv** - Legen Sie fest, ob die Abwesenheitsart aktiv ist
4. Klicken Sie auf **Speichern**.

## Abwesenheitsart bearbeiten

1. Klicken Sie auf eine Abwesenheitsart in der Liste.
2. Passen Sie die gewünschten Felder an.
3. Klicken Sie auf **Speichern**.

## Abwesenheitsart löschen

1. Klicken Sie auf eine Abwesenheitsart in der Liste.
2. Klicken Sie auf **Löschen**.
3. Bestätigen Sie den Löschvorgang.

## Prozentuale Anrechnung

Die prozentuale Anrechnung bestimmt, wie stark eine Abwesenheit vom Kontingent abgezogen wird:
- **100%** - Volle Anrechnung (z. B. regulärer Urlaub)
- **0%** - Keine Anrechnung (z. B. Fortbildung)
- **50%** - Halbe Anrechnung (z. B. Sonderurlaub)

## Mitarbeiter-Berechtigungen

Sie können festlegen, ob Mitarbeiter die Abwesenheitsart selbst beantragen dürfen:
- **Ja** - Mitarbeiter können die Abwesenheit selbst beantragen
- **Nein** - Nur Vorgesetzte oder Administratoren können diese Abwesenheitsart anlegen
- **Mit Genehmigung** - Mitarbeiter können beantragen, benötigen aber eine Genehmigung

## Farbkennzeichnung

Die gewählte Farbe wird in der Kalenderansicht und in Übersichten verwendet, um verschiedene Abwesenheitsarten schnell unterscheiden zu können.

> **Hinweis:** Bevor Sie eine Abwesenheitsart löschen, prüfen Sie, ob bereits Abwesenheitsanträge mit dieser Art existieren. Inaktive Abwesenheitsarten können nicht mehr für neue Anträge verwendet werden, bleiben aber in bestehenden Anträgen erhalten.
