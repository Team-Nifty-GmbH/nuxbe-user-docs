# Kontakte zusammenführen

Wenn in Ihrem System doppelte Kontakte vorhanden sind, können Sie diese zusammenführen. Beim Zusammenführen werden alle Daten (Adressen, Aufträge, Kommunikation, Anhänge etc.) des zusammenzuführenden Kontakts auf den Hauptkontakt übertragen.

## Voraussetzungen

- Sie benötigen die entsprechende Berechtigung zum Zusammenführen von Kontakten.
- Der Vorgang kann **nicht rückgängig** gemacht werden. Prüfen Sie vorab sorgfältig, welchen Kontakt Sie als Hauptkontakt behalten möchten.

## Kontakte zusammenführen

1. Navigieren Sie zur [Kontaktliste](1-kontakte-verwalten.md).

2. Wählen Sie die Kontakte aus, die Sie zusammenführen möchten, indem Sie die Kontrollkästchen in der ersten Spalte aktivieren.

3. Klicken Sie auf die Schaltfläche **Zusammenführen** in der Aktionsleiste oberhalb der Tabelle.

4. Im Auswahlfenster sehen Sie die ausgewählten Kontakte nebeneinander. Wählen Sie den **Hauptkontakt** -- das ist der Kontakt, der nach dem Zusammenführen bestehen bleibt.

5. Prüfen Sie die Vorschau, welche Daten übertragen werden.

6. Klicken Sie auf **Zusammenführen**, um den Vorgang zu bestätigen.

Nach dem Zusammenführen:

- Der **Hauptkontakt** bleibt bestehen und enthält alle zusammengeführten Daten.
- Alle **anderen Kontakte** werden gelöscht.
- Sämtliche Verknüpfungen (Aufträge, Adressen, Kommunikation, Projekte, Tickets, Anhänge) werden auf den Hauptkontakt übertragen.

> **Wichtig:** Wählen Sie den richtigen Kontakt als Hauptkontakt. Wenn Sie versehentlich den falschen Kontakt auswählen, gehen die Stammdaten des eigentlich gewünschten Hauptkontakts verloren. Der Vorgang kann nicht ohne weiteres rückgängig gemacht werden.

## Empfohlener Ablauf

1. **Vor dem Zusammenführen:** Öffnen Sie beide Kontakte in separaten Tabs und vergleichen Sie die Stammdaten (Kundennummer, Zahlungsart, Preisliste etc.). Notieren Sie sich, welcher Kontakt die korrekten Stammdaten enthält.

2. **Hauptkontakt wählen:** Wählen Sie den Kontakt als Hauptkontakt, dessen Stammdaten korrekt sind und beibehalten werden sollen.

3. **Nach dem Zusammenführen:** Prüfen Sie den resultierenden Kontakt und korrigieren Sie gegebenenfalls einzelne Felder.
