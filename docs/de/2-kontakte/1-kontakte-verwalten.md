# Kontakte verwalten

In der Kontaktliste sehen Sie alle gespeicherten Kontakte auf einen Blick. Die Liste bietet umfangreiche Funktionen zum Suchen, Filtern und Sortieren.

## Kontaktliste öffnen

1. Klicken Sie in der Sidebar auf **Kontakte**.

   ![Kontaktliste mit allen Kontakten](../screenshots/06-kontakte-liste.png)

2. Die Tabelle zeigt alle Kontakte mit folgenden Spalten:
   - **Avatar** - Profilbild oder Initialen des Kontakts
   - **Kundennummer** - Eindeutige Nummer des Kontakts
   - **Name** - Firmenname oder Personenname
   - **Debitorennummer** - Debitorennummer für die Buchhaltung
   - **Kreditorennummer** - Kreditorennummer für die Buchhaltung (bei Lieferanten)
   - **Zahlungsart** - Hinterlegte Standard-Zahlungsart
   - **Zahlungsziel** - Vereinbartes Zahlungsziel in Tagen

## Kontakte suchen

Oberhalb der Tabelle befindet sich ein Suchfeld. Geben Sie dort einen Begriff ein, um Kontakte zu finden. Die Suche durchsucht alle sichtbaren Felder gleichzeitig, z. B. Name, Kundennummer, Debitorennummer oder Kreditorennummer. Die Ergebnisse werden in Echtzeit aktualisiert, während Sie tippen.

## Spalten sortieren

Klicken Sie auf eine Spaltenüberschrift, um die Tabelle nach dieser Spalte zu sortieren. Ein erster Klick sortiert aufsteigend (A-Z bzw. kleinste Zahl zuerst). Ein zweiter Klick sortiert absteigend (Z-A bzw. größte Zahl zuerst).

Die aktive Sortierung wird durch einen Pfeil neben der Spaltenüberschrift angezeigt.

## Filter verwenden

### Spaltenfilter

Unterhalb der Spaltenüberschriften befindet sich eine Filterzeile. Dort können Sie für jede einzelne Spalte einen Filterwert eingeben. So schränken Sie die Ergebnisse gezielt ein, z. B. nur Kontakte mit einer bestimmten Zahlungsart.

Mehrere Spaltenfilter lassen sich gleichzeitig verwenden. Die Ergebnisse werden auf Einträge reduziert, die alle gesetzten Filter erfüllen.

### Gespeicherte Filter

Über die Schaltfläche **Gespeicherte Filter** rufen Sie zuvor gespeicherte Filterkonfigurationen auf. Damit können Sie häufig verwendete Filtereinstellungen mit einem Klick wiederherstellen.

So speichern Sie einen Filter:

1. Setzen Sie die gewünschten Spaltenfilter und Suchbegriffe.
2. Klicken Sie auf **Gespeicherte Filter**.
3. Vergeben Sie einen Namen für den Filter.
4. Klicken Sie auf **Speichern**.

Der gespeicherte Filter steht Ihnen ab sofort in der Filterliste zur Verfügung.

## Paginierung

Am unteren Rand der Tabelle finden Sie die Seitennavigation. Dort sehen Sie die aktuelle Seitennummer und die Gesamtanzahl der Seiten. Sie können zwischen den Seiten blättern oder eine bestimmte Seite direkt ansteuern.

Zusätzlich können Sie die Anzahl der angezeigten Einträge pro Seite anpassen. Wählen Sie dazu den gewünschten Wert aus dem Dropdown neben der Seitennavigation.

## Neuen Kontakt anlegen

1. Klicken Sie auf die Schaltfläche **Neu** oben rechts.
2. Die Detailansicht für einen neuen Kontakt öffnet sich.
3. Füllen Sie die gewünschten Felder aus. Pflichtfelder sind mit einem Stern gekennzeichnet.
4. Im Tab **Allgemein** hinterlegen Sie die Stammdaten wie Mandant, Zahlungsart, Zahlungsziel, Preisliste und Rabattgruppe.
5. Wechseln Sie zum Tab **Adressen**, um mindestens eine Adresse mit Firmenname, Straße, PLZ, Ort und Land anzulegen.
6. Im Tab **Kommunikation** können Sie E-Mail-Adressen, Telefonnummern und weitere Kontaktdaten hinterlegen.
7. Klicken Sie auf **Speichern**, um den neuen Kontakt zu erstellen.

Nach dem Speichern werden Kundennummer, Debitorennummer und ggf. Kreditorennummer automatisch vergeben, sofern Sie diese nicht manuell eingetragen haben.

## Kontakt löschen

1. Öffnen Sie die Detailansicht des Kontakts.
2. Klicken Sie auf **Löschen**.
3. Bestätigen Sie den Löschvorgang im angezeigten Dialog.

Gelöschte Kontakte werden per Soft Delete entfernt und können bei Bedarf wiederhergestellt werden.
